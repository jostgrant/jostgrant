### DESCRIPTION

Hey fellers & fellas, Grant here & welcome to my Github! The scope of this account has changed a lot during the past year or-so -- like many, the events of the 2020 - 2021(+ ?) COVID-19 Pandemic has caused me to [reevaluate my life](https://jostgrant.github.io/foreword-on-moving-forwards/) in some nontrivial ways. This profile has become the manifestation of that lingering affect by working towards **actually** being 'effective' in the world over any moralizing, pleasure-seeking, and/or pain avoidance that I've so readily fallen into. So the "goal" for this space, between now and 2023 / 2024, is to build some kind of project portfolio in-line with this ethos.
